About this site

Publishing

  • This is a static site built using hakyll, and hosted via GitHub pages.
  • I support the movement to decentralize the web; as such, you can also access this site via modern peer-to-peer protocols.
    • You can find me via dat @ dat://patrl.hashbase.io, using, e.g., beaker browser.
    • You can find me on the IPFS network at ipns/patrickdelliott.com. If you use the IPFS companion browser extension, then you should be automatically redirected.

Design

  • This site is built to be fast and responsive using a custom theme built with the tachyons css framework - it is 100% javascript free. The theme is tailored to style html generated by pandoc.
  • The font is bold monday’s beautiful IBM Plex typeface.

Content

  • The vast majority of the content is written in emacs org-mode, and converted to html5 via pandoc’s built-in org-mode reader.
